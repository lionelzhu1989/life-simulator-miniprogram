## 五、永久行为规则

1. **🚫 禁止用 Python 脚本批量替换 `<script>` 内的 JS 代码**
   → 这是 A/E 类最大来源（6/16 例）。改用 `read_file → edit_file` 单行精准编辑。

2. **每次 HTML/JS 修改后必须执行 L1.1+L1.2+L1.3**

   → 防止改了路径分支导致 emoji 消失。

5. **每次 commit 前 L1-L3 全部通过，输出记录附在 commit message 中**
