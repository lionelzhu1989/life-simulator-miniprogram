# 人生首富模拟器 微信小程序

一款面向校园群体的虚拟人生模拟小程序，融合电子宠物养成、模拟炒股、炒房投资、职业成长玩法，可作为校园招聘AI产品的引流裂变工具。

## 功能特性
- 🧑💼 职业成长系统：多档位职业入职，属性门槛匹配
- 📈 模拟股市：多支股票随机行情，完整买卖交易
- 🏠 房产系统：全款/按揭贷款买房，租金收益，逾期拍卖
- 👨‍👩‍👧 家庭系统：结婚生子，配偶薪资加成，子女抚养
- 🎲 随机事件：每年概率触发好事/坏事/日常事件
- 🏆 校园财富排行榜：本地模拟排行，支持好友PK分享
- 🎯 内置校招AI跳转入口，可直接对接主招聘产品

## 快速运行
1. 安装「微信开发者工具」
2. 新建小程序项目，选择本项目目录
3. 填入测试AppID，选择「不使用云服务」
4. 点击编译即可预览运行

## 二次开发说明
- 游戏数值配置：`utils/gameConfig.js`
- 核心结算逻辑：`utils/gameEngine.js`
- 校招AI对接：`pages/index/index.js` 中 `goAI` 方法，替换为目标小程序AppID
- 所有页面数据均从全局 `app.globalData.player` 读取，统一存档到本地缓存

## 合规声明
本程序所有金融、房产、股票数据均为虚拟数值，仅用于游戏娱乐，不构成任何现实投资建议。
